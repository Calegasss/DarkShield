# Dark Shield

GitHub Pages-ready WebGL build for **Dark Shield**.

## Play online

After GitHub Pages is enabled for this repository, the game will be available at:

`https://calegasss.github.io/DarkShield/`

## Project structure

- `docs/`: static site deployed by GitHub Pages

## Publishing updates

The raw Unity WebGL export at the repository root is intentionally ignored by Git.
The `docs/` folder contains the version that is compatible with GitHub Pages, using
uncompressed WebGL files so the site works without custom server headers.
